# Obsidian.md-Custom-CSS
Custom CSS for Obsidian.md

# Search Highlight
Adds a blue background to text-search results to accomodate for visibility in some themes

# Callouts - outline style
Adds outline-style callouts, used by adding callouts  
`> [!outline-red]`  
`> [!outline-orange]`  
`> [!outline-yellow]`  
`> [!outline-green]`  
`> [!outline-blue]`  
`> [!outline-purple]`  
`> [!outline-cyan]`  
`> [!outline-pink]`  

Works on Light and Dark theme, using theme-specified colors. Title currently only shows in "Edit" mode. Refinements coming to fix "Reading" mode.
